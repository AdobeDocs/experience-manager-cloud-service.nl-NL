# Opmerkingen bij de onderhoudsrelease {#maintenance-release-notes}

In het volgende gedeelte worden de opmerkingen bij de technische release voor de onderhoudsreleases van Experience Manager as a Cloud Service beschreven die zijn gekoppeld aan activering van de functie 2024.11.0.

## Release 18598 {#18598}

Hieronder worden de voortdurende verbeteringen voor onderhoudsrelease 18598 samengevat, die op 13 november 2024 openbaar werd gemaakt. De vorige onderhoudsrelease was release 18311. Release 18459 is privé gemaakt vanwege een uitgave.

De activering van de 2024.11.0-functie biedt de volledige functionaliteit die is ingesteld voor deze onderhoudsrelease. Zie [&#x200B; Experience Manager geeft Roadmap &#x200B;](https://experienceleague.adobe.com/en/docs/experience-manager-release-information/aem-release-updates/update-releases-roadmap) voor meer informatie vrij.
